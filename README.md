# HotSpot

Este repositorio contiene una configuración completa para desplegar un punto de acceso WiFi funcional basado en Linux. Incluye la integración y configuración de servicios esenciales como "hostapd" para la gestión del punto de acceso inalámbrico, "dnsmasq" para el servicio de DNS y DHCP ligero, y "nodogsplash" para la implementación de un portal cautivo.

Además, se incorporan archivos de configuración de red (WLAN), reglas de DHCP personalizadas y la configuración de "Apache" y "PHP" como servidor web para alojar la interfaz del portal cautivo o páginas informativas.

El objetivo de este proyecto es proporcionar una solución lista para usar que permita crear una red WiFi controlada, ideal para entornos como laboratorios, redes públicas o proyectos educativos relacionados con redes y ciberseguridad.
